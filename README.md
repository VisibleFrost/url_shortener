Асинхронный сервис для сокращения ссылок, построенный на базе FastAPI с
использованием архитектурных паттернов кэширования для работы под высокой нагрузкой.

Для использования нужно иметь Docker и Docker Compose.

Запустите с помощью команды: 
```bash
docker compose up --build
```
После этого зайдите на http://localhost:8000/docs и в POST-эндпоинте можете вставить длинную ссылку и получить ее сокращение.
В GET-роуте можно перейти по готовой ссылке, если браузер позволит.
